1: Install UE4SS under:  <Palworld>/Pal/Binaries/Win64 - https://pwmodding.wiki/docs/users/ue4ss/downloading
2: Copy this folder to:  .../ue4ss/Mods/PalworldEasyCheats
3: Run PalworldEasyCheats.bat
4: Load a world and use hotkeys. Restart Palworld after changing menu options or refresh mods in UE4SS.



HotKeys & Toggles - PalworldEasyCheatsMenu.bat

-- Hotkeys (in game) --
+ / -     Character speed up / down
Alt+0 -   Reset character speed
Alt+J -   Toggle high jump
Alt+F -   Toggle unlimited fuel
Alt+S -   Toggle unlimited stamina
Alt+G -   Toggle god mode + full heal
Alt+H -   Toggle disable hunger

-- Permanent --
0) Reveal Full Map

-- On / Off --
1) Skip Mod Disclaimer
2) High Jump
3) Sync Swim Speed
4) Infinite Fuel
5) Infinite Stamina
6) God Mode On Start
7) Include Pals (God Mode)
8) Disable Hunger On Start
9) Include Pals (Hunger)

-- Values --
10) Character Speed
11) High Jump Height

12) Reset Everything To Normal

Restart Palworld after changing menu options or refresh mods in UE4SS.
